## **Descripción del Proyecto**
Este proyecto corresponde al desarrollo de los módulos del sistema **Montecafe**, el cual permite la gestión de clientes, proveedores, inventario, materia prima y ventas.  
El sistema se construyó utilizando **Java EE (Servlets y JSP)**, siguiendo las buenas prácticas de programación y los lineamientos del SENA para el ciclo de desarrollo del software.

El sistema incluye:
- Formularios HTML conectados con Servlets y JDBC.
- Métodos HTTP **GET** y **POST**.
- Páginas JSP dinámicas con conexión a base de datos.
---

## **Módulos Desarrollados**
- **Login:** pendiente Validación de usuario
- **Dashboard:** Panel principal con indicadores/pendiente la navegación entre modulos.
- **Clientes:** Registro y listado de clientes.
- **Proveedores:** Administración de proveedores.
- **Materia Prima:** Gestión de insumos.
- **Inventario:** Control de productos y stock.
- **Ventas:** Registro y visualización de ventas.

---

## **Arquitectura y Tecnologías**
- **Lenguaje:** Java (Servlets, JSP, JDBC)
- **Servidor:** Apache Tomcat 9+
- **Base de Datos:** MySQL 8+
- **Frontend:** HTML5, CSS3, JavaScript
- **IDE:** NetBeans
- **Versionamiento:** Git y GitHub
- **Gestión de dependencias:** Maven
